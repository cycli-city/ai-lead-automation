# AI Lead Automation System 🤖

An end-to-end AI-powered lead capture and voice calling system built with no code.

## 🎯 What it does
- Captures leads from a Tally form
- Creates CRM record in Airtable automatically
- Sends confirmation email via Gmail
- Triggers AI voice call via VAPI within 60 seconds
- AI agent books appointments in natural conversation

## 🛠 Tech Stack
- **n8n** — workflow automation
- **VAPI** — AI voice agent
- **Airtable** — CRM database
- **Twilio** — phone infrastructure
- **Groq / Llama 3.3** — AI language model
- **Tally** — lead capture form
- **Gmail** — email confirmation

## 🔄 System Flow
- Lead fills Tally form
- n8n webhook receives data
- Airtable CRM record created
- Gmail confirmation sent
- VAPI triggers AI outbound call
- Sarah books appointment
